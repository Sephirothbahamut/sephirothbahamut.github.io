Content and Features

GraphMaker is a program which lets you build simple graphs, (oriented, non-oriented or even mixed) with ease. (on bottom a video showing BFS after 22/02/2017 update)

All of program functionalities are 100% completed, but as you may notice, top-bar and left-bar excluded, the user interface is not much "cured" and only features default look. Since the GUI library is going to get substantial modifications during my future revision, GraphMaker's aesthetics will remain in stand-by till the new (hopefully good enough to be published) version of the gui will be released. Then the very first thing i will do is updating GraphMaker using that version, which grants improved performances to the software itself, and i will give it some more cured aesthetics as well.

Please if you can test the program and let me know of any bug, especially during the visits simulations (providing .grph file of the graph giving error messages if you can)

Current State

  • Add as many nodes and arcs between nodes as you wish. Multiple arcs between the same two nodes are supported.
  • Move selected items around with grid snapping.
  • Customize grid snap size for both axis (x and y) with the value you prefer.
  • Swap directions of oriented arcs
  • Highlight all arcs and nodes which can be reached from a certain node
  • Save/open graphs
  • Export graphs as .txt files in various formats:
    • Adjacency vector
    • Adjacency matrix
    • Arcs list
  • Execute an animated visit simulation with custom time delays
    • DFS
    • BFS
  • Customize colors for nodes, arcs, visited items, selected items, background, and grid.
  • Add/show/hide up to three custom variables to your nodes and/or your arcs.
  • Change custom variable's value dynamically when executing a simulated visit, like set/increase/decrease a certain variable by a certain value.
  • Add conditions to custom variables changing during simulated visits. (suspended, partially available)
