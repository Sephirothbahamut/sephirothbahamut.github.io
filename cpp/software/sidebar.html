C++ > Windows SideBar

Content and Features

As the name suggests, this software is strictly Windows only, Linux version is not planned.

Windows SideBar is an highly customizable desktop tool, which adds a bar on the right side of the screen. The bar has an analog clock, which Windows 10 lacks of by default, and a customizable amount of buttons.

The buttons can either run a program, open a website, or even run a console command.

Colors can be customized.

By default an additional decoration is active which renders some nice particle effects behind your taskbar; if you don't see the particles, you have to enable taskbar transparency.

Current State

The program itself has been completed. Right now it only lacks the following extra features:
  • Customization and settings window
  • Drag&Drop links and programs to automatically add a button
  • Support for different taskbar positions
