Advanced Getline

Content and Features

The purpose of advanced getilne is to make a c++ getline feel like a regular textbox to the user.
That means the user will be able to move the cursor, highlight text, overwrite, access memory of previous inputs, delete parts of the text, and various other things.
Moreover the developer can further customized getline's colors to make them fit the console in case the software getline is being called from makes use of console colors modifications.

Current State

The program is almost completed in the Windows version, although it still lacks some functionality like insert and copy-paste. Linux version is almost ready to be added to the repository but it's not there yet. The current version, despite lacking some functionalities, is already fully usable and i'd like people to actually use it and maybe report existing bugs i didn't find.
