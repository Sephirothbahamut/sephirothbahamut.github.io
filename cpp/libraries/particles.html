Particle System

Attention:

This project is likely to be abandoned.

This is my first attempt at creating a particles effect in c++
It's fixed-step dependent, but can be made dynamic easily. In this system, particles are supposed to be created in random positions within a certain area, called "generator". You can set the base values for various particles properties, as well as an offset which will be used to randomize each single particle.

These are the properties which can be customized:
  • Size scale (both x and y)
  • Base speed
  • Acceleration/deceleration over time
  • Lifetime (in steps)
  • Direction

Each property can have an offset which will used to randomize values. For example, if base speed is 10, and the offset is 2, the particle generator will create particles which speed is equal to 8-12 units per step.
